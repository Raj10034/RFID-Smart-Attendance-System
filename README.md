# 📡 **RFID Smart Attendance System**

The RFID Smart Attendance System is an IoT-based solution that automates attendance tracking using the NodeMCU ESP8266 and RFID RC522. When a user scans an RFID card, the system processes the UID and sends it wirelessly to a Python backend, where attendance records are stored in CSV/MongoDB. A simple web dashboard allows administrators to view real-time logs and daily attendance reports.

**RFID Attendance | NodeMCU ESP8266 | Python Backend | MongoDB | HTML/CSS/JS | IoT Project**

# 📌 **Project Overview**

* IoT-based smart attendance solution
* Uses **NodeMCU ESP8266 + RFID RC522** to read RFID card UIDs
* Sends attendance data wirelessly to a **Python backend**
* Stores logs in **CSV/MongoDB**
* Web dashboard provides real-time logs & daily reports

# 🏗 **Workflow**

1. User scans RFID card
2. RC522 reads RFID UID
3. NodeMCU timestamps and sends data over WiFi
4. Python backend logs attendance
5. Dashboard displays real-time attendance
6. Admin exports reports
